Forge 1.20.6 uses the server-side Snow Accumulation implementation in src/main/java.
The mod runs from a server tick listener, hot-reloads config/snowaccumulation/config.txt, and grows snow one layer at a time.
This folder exists so the scaffold can apply the correct Forge userdev dependencies and Java level for 1.20.6. Exact Forge 1.20.5 is intentionally excluded because this scaffold only supports 1.20.6 in the late 1.20 family.
