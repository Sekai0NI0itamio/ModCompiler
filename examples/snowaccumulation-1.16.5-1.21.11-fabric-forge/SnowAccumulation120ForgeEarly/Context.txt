Forge 1.20.1-1.20.4 uses the server-side Snow Accumulation implementation in src/main/java.
The mod runs from a server tick listener, hot-reloads config/snowaccumulation/config.txt, and grows snow one layer at a time.
This folder exists so the scaffold can apply the correct Forge userdev dependencies and Java level for 1.20.1-1.20.4.
