Fabric 1.20.1-1.20.6 uses the server-side Snow Accumulation implementation in src/main/java.
The mod polls only the overworld on END_SERVER_TICK, hot-reloads config/snowaccumulation/config.txt, and grows snow one layer at a time.
This folder exists so the scaffold can apply the correct Fabric dependencies and Java level for 1.20.1-1.20.6.
