General

Welcome to Auaci — a lightweight developer environment that blends an editor, project browser, and chat-powered assistant into one workspace. This settings area will let you configure global behavior, appearance, and integrations. For now, this page provides a short overview of the application and its goals.

What Auaci offers

  • Fast file browsing and editing across your project directory.
  • Chat-driven code assistance and automated suggestions.
  • Extensible components and a focused UI for coding workflows.

This settings UI will be expanded with toggles, profiles, and integration options. You can expect granular control over features, theme options, and privacy-related settings.
