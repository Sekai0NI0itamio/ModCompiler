## Features
- Automatically switches to the best tool when mining blocks
- Smart tool detection: shovel for dirt/sand, axe for wood, pickaxe for everything else
- Enchantment preferences (Fortune, Silk Touch, Efficiency, Unbreaking)
- Switch back to previous item after mining (2 second delay, configurable)
- Hotbar-only mode or full inventory search
- Fully configurable via config file
- Client-side only for smooth visual sync
