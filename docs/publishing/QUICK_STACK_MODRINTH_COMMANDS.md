## Features
- Quick stack items to nearby containers with V key
- Only stacks items that already exist in containers (smart matching)
- Hotbar protection (items in hotbar are not moved by default)
- Configurable search range (default 8 blocks, max 16)
- Sound feedback when items are stacked
- On-screen message showing what was stacked
- Works with chests, trapped chests, barrels, and shulker boxes
- Fully configurable via config file
- Client-server synchronization for multiplayer support

## Configuration Options
```
enableQuickStack (default: true) - Enable/disable Quick Stack feature
searchRange (default: 8, range: 1-16) - Range in blocks to search for containers
includeHotbar (default: false) - Include hotbar items in quick stack
playSound (default: true) - Play sound feedback
showMessage (default: true) - Show on-screen message
stackToChests (default: true) - Stack to chests and trapped chests
stackToBarrels (default: true) - Stack to barrels
stackToShulkerBoxes (default: true) - Stack to shulker boxes
```

## Key Binding
- Default: V key
- Customizable in Minecraft Options → Controls → Key Binds → Inventory → "Quick Stack"

## How It Works
1. Client presses V key
2. Client sends packet to server
3. Server scans nearby containers (within configured range)
4. Server moves matching items from player inventory to containers
5. Server sends feedback (sound + message) to client

## Technical Implementation
- Uses Forge SimpleNetworkWrapper for client-server communication
- Server-side inventory operations for multiplayer compatibility
- Smart item matching using ItemStack comparison
- Configurable container type filtering
