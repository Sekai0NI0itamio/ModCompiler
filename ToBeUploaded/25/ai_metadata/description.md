# Auto Crystal

Automatically attacks end crystals when you place them! Perfect for PvP crystal combat.

## Features

- ✅ **Automatic Attack** - Instantly attacks crystals when you place them
- ✅ **0-Tick Speed** - Instant attack by default (configurable)
- ✅ **Toggle Control** - Press C to enable/disable
- ✅ **Configurable Range** - Adjust attack distance (default 6 blocks)
- ✅ **Visual Feedback** - Chat messages show ON/OFF status
- ✅ **PvP Optimized** - Designed for crystal PvP combat

## How It Works

1. Hold an end crystal in your hand
2. Place it on obsidian or bedrock
3. The mod automatically attacks it to explode it
4. No manual clicking required!

## Controls

- **C Key**: Toggle Auto Crystal on/off

## Configuration

Config file: `config/autocrystal.cfg`

- `enabled`: Enable/disable the mod (default: true)
- `attackDelay`: Delay in ticks before attacking (default: 0 = instant)
- `attackRange`: Maximum range to attack crystals (default: 6.0 blocks)

## PvP Usage

This mod is designed for crystal PvP combat:
- Place crystals near opponents for instant explosions
- Faster reaction time than manual clicking
- Toggle off when not in combat
- Configurable for different playstyles

## Installation

1. Download the mod JAR file
2. Place it in your `.minecraft/mods` folder
3. Launch Minecraft 1.12.2 with Forge installed
4. Press C to toggle the mod on/off

## Compatibility

- **Client-side**: Required (client-only mod)
- **Server-side**: Not needed (works on any server)
- **Mod Compatibility**: Works with most mods

## Performance

Lightweight and efficient:
