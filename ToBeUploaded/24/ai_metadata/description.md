# Better Sprinting

**Stop holding keys! Toggle sprint and sneak with one press.**

Better Sprinting is the ultimate movement enhancement mod that lets you toggle sprint and sneak instead of holding keys. Sprint in all directions, boost your flying speed, and see your status on the HUD. Simple, lightweight, and essential for every player.

## ✨ Key Features

### Toggle Sprint
- ✅ **One Press Toggle** - Press sprint key once, start sprinting
- ✅ **All Directions** - Sprint forward, backward, sideways
- ✅ **Disable Double-Tap** - No more accidental sprinting
- ✅ **Vanilla Key** - Uses your normal sprint key

### Toggle Sneak
- ✅ **One Press Toggle** - Press sneak key once, start sneaking
- ✅ **No Holding** - Free up your pinky finger
- ✅ **Vanilla Key** - Uses your normal sneak key

### Flying Boost
- ✅ **Sprint to Boost** - Sprint key boosts flying speed
- ✅ **Configurable** - 1x to 5x speed multiplier
- ✅ **Creative Mode** - Perfect for building

### HUD Indicators
- ✅ **Sprint Status** - Shows "[Sprint: ON]" in green
- ✅ **Sneak Status** - Shows "[Sneak: ON]" in yellow
- ✅ **Configurable Position** - Move anywhere on screen
- ✅ **Toggle Display** - Can be disabled

## 🎮 How to Use

### Toggle Sprint
1. Press your sprint key (default: Left Control)
2. Start moving - you're now sprinting!
3. Press sprint key again to turn off

**Sprint in all directions:**
- Forward: Sprint normally
- Backward: Sprint while moving back
- Sideways: Sprint while strafing
- Any combination works!

### Toggle Sneak
1. Press your sneak key (default: Left Shift)
2. You're now sneaking!
3. Press sneak key again to turn off

### Flying Boost
1. Enter creative/spectator mode
2. Start flying
3. Press sprint key (or toggle it on)
4. Zoom through the sky at 2x speed (configurable)!

## ⚙️ Configuration

Config file: `config/bettersprinting.cfg`

### Sprint Settings
```
Enable Toggle Sprint (default: true)
- Enable toggle sprint feature

Sprint In All Directions (default: true)
- Allow sprinting in any direction (not just forward)

Disable Double Tap (default: true)
- Disable vanilla double-tap to sprint
- Only use toggle key
```

### Sneak Settings
```
Enable Toggle Sneak (default: true)
- Enable toggle sneak feature
```

### Flying Settings
```
Enable Fly Boost (default: true)
- Enable sprint key to boost flying speed

Fly Boost Amount (default: 2.0, range: 1.0-5.0)
- Flying speed multiplier
- 1.0 = normal speed
- 2.0 = double speed
- 5.0 = 5x speed!
```

### HUD Settings
```
Show Sprint HUD (default: true)
- Show sprint status on screen

Show Sneak HUD (default: true)
- Show sneak status on screen

HUD X Position (default: 5)
- Horizontal position (pixels from left)

HUD Y Position (default: 5)
- Vertical position (pixels from top)
```

## 📋 FAQ

**Q: Does this work on servers?**  
A: Yes! It's client-side only, so it works everywhere.

**Q: Can I still use vanilla sprint?**  
A: Yes! You can disable toggle sprint and use vanilla if you want.

**Q: Does this work with other movement mods?**  
A: Yes! Compatible with most mods.

**Q: Can I sprint backward?**  
A: Yes! Enable "Sprint In All Directions" in config.

**Q: How do I turn off the HUD?**  
A: Set "Show Sprint HUD" and "Show Sneak HUD" to false in config.

**Q: Can I change the fly boost speed?**  
A: Yes! Set "Fly Boost Amount" from 1.0 to 5.0 in config.

**Q: Does this work in creative mode?**  
A: Yes! Perfect for creative building with fly boost.

**Q: Will this get me banned on servers?**  
A: No! It's a client-side QoL mod. Check server rules to be sure.
