GUARANTEED PLAYER-VISIBLE FACTS
- Shows “Day X” where X is the current day number based on world time.
- Optionally shows the current time of day (hour and minute) in 12‑hour format with AM/PM.
- Updates the displayed day number automatically as the world progresses.
- Reads a config file that can be edited while the game runs; changes appear after a short delay.
- Positions the text according to an anchor (default top‑right) and offset pixels.
- Only draws the overlay when the player is in the world and the GUI is not hidden.
- Works only on the client side; no commands, items, or blocks are added.

- Unclear if the mod functions on multiplayer servers.
- Unclear whether editing the config file while the game is paused takes effect immediately.
- Unclear if other anchor positions or display modes are supported beyond those listed.
