# TNT Duper

Allows dispensers to duplicate TNT without using up the TNT item.

### What It Does (for the user)

- Right‑click a dispenser that contains TNT and it will spawn a primed TNT entity.
- The TNT stays in the world and can be picked up or explode normally.
- The TNT item used to trigger the dispenser remains in the dispenser, so you can keep using it.
- Only the priming sound plays; no extra animation or consumption occurs.

### Features

- Infinite TNT dispensing from a single TNT item.
- Works with any dispenser that holds TNT.
- Compatible with vanilla TNT mechanics (same explosion behavior).
- Works on both single‑player and multiplayer worlds.

### Why Use This Mod?

- Build automatic TNT farms without constantly refilling dispensers.
- Simplify redstone contraptions that need repeated TNT bursts.
- Save time and resources when testing explosions.
- Easy to use – just place a dispenser and watch it duplicate TNT.

## What You'll Notice

- Dispensers can fire TNT without using up the TNT item.
- The spawned TNT behaves like normal TNT and can be picked up or detonated.

package com.forgetemplatemod;

import net.minecraft.block.BlockDispenser;
import net.minecraft.dispenser.BehaviorDefaultDispenseItem;
import net.minecraft.dispenser.IBlockSource;
import net.minecraft.entity.item.EntityTNTPrimed;
import net.minecraft.init.SoundEvents;
import net.minecraft.item.ItemStack;
import net.minecraft.util.SoundCategory;
import net.minecraft.util.math.BlockPos;
import net.minecraft.world.World;

public class BehaviorDispenseTNTDuper extends BehaviorDefaultDispenseItem {
    @Override
    protected ItemStack dispenseStack(IBlockSource source, ItemStack stack) {
        World world = source.getWorld();
        BlockPos blockpos = source.getBlockPos().offset(source.getBlockState().getValue(BlockDispenser.FACING));
        EntityTNTPrimed entitytntprimed = new EntityTNTPrimed(world, (double)blockpos.getX() + 0.5D, (double)blockpos.getY(), (double)blockpos.getZ() + 0.5D, null);
        world.spawnEntity(entitytntprimed);
        world.playSound(null, entitytntprimed.posX, entitytntprimed.posY, entitytntprimed.posZ, SoundEvents.ENTITY_TNT_PRIMED, SoundCategory.BLOCKS, 1.0F, 1.0F);

        // Return the stack unchanged! We intentionally DO NOT call stack.shrink(1)
        // This makes the dispenser act as a TNT Duper without consuming the item.
        return stack;
    }
}
