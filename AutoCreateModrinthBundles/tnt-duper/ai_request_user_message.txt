# ServerCore

## Description

**ServerCore** is a lightweight utility mod for Minecraft Forge 1.12.2 that adds a complete, player-friendly teleport toolkit to both singleplayer and multiplayer—without requiring cheats or operator status. Whether you're exploring, building, or just getting started, ServerCore gives you the tools to move around your world with ease.

### What It Does (for the user)

**Teleport Requests**
- Send a teleport request to another player:  
  `/tpa <player>` – Request to teleport to them.  
  `/tpahere <player>` – Request them to teleport to you.
- Incoming requests last 60 seconds before expiring.
- Accept or deny requests individually or all at once:  
  `/tpaccept [player]` – Accept a specific request.  
  `/tpacceptall` – Accept all pending requests.  
  `/tpadeny [player]` – Deny a specific request.  
  `/tpadenyall` – Deny all pending requests.
- Cancel requests you've sent:  
  `/tpacancel <player>` – Cancel a specific outgoing request.  
  `/tpacancel all` – Cancel all outgoing requests.

**Personal Homes**
- Set your own named homes that remember your exact location, dimension, and even which way you're facing:  
  `/sethome <name>` – Save your current position as a home.
- Teleport to any of your homes anytime:  
  `/home <name>` – Go to that home.  
  `/home list` – See all your saved homes.  
  `/delhome <name>` – Remove a home.
- Homes are private to you and persist across sessions in the world data.

**Safe Random Teleport**
- Instantly teleport to a random, safe location without leaving your dimension:  
  `/rtp overworld` – Random location in the Overworld.  
  `/rtp nether` – Random location in the Nether.  
  `/rtp end` – Random location in the End.
- The search area is a 10,000-block square centered at 0,0 (respecting your world border).
- The mod ensures you land on a safe spot—no mid-air or suffocation spawns.

**First-Join Auto-Teleport**
- When a player joins a world for the very first time, they are automatically teleported to a safe random overworld location (using the same RTP system).
- Perfect for survival servers: new players start somewhere fresh and unexplored, not at world spawn.

package com.forgetemplatemod;

import net.minecraft.block.BlockDispenser;
import net.minecraft.dispenser.BehaviorDefaultDispenseItem;
import net.minecraft.dispenser.IBlockSource;
import net.minecraft.entity.item.EntityTNTPrimed;
import net.minecraft.init.SoundEvents;
import net.minecraft.item.ItemStack;
import net.minecraft.util.SoundCategory;
import net.minecraft.util.math.BlockPos;
import net.minecraft.world.World;

public class BehaviorDispenseTNTDuper extends BehaviorDefaultDispenseItem {
    @Override
    protected ItemStack dispenseStack(IBlockSource source, ItemStack stack) {
        World world = source.getWorld();
        BlockPos blockpos = source.getBlockPos().offset(source.getBlockState().getValue(BlockDispenser.FACING));
        EntityTNTPrimed entitytntprimed = new EntityTNTPrimed(world, (double)blockpos.getX() + 0.5D, (double)blockpos.getY(), (double)blockpos.getZ() + 0.5D, null);
        world.spawnEntity(entitytntprimed);
        world.playSound(null, entitytntprimed.posX, entitytntprimed.posY, entitytntprimed.posZ, SoundEvents.ENTITY_TNT_PRIMED, SoundCategory.BLOCKS, 1.0F, 1.0F);

        // Return the stack unchanged! We intentionally DO NOT call stack.shrink(1)
        // This makes the dispenser act as a TNT Duper without consuming the item.
        return stack;
    }
}
