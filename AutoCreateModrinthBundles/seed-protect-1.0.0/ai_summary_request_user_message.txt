# Seed Protect

This mod stops mobs and players from trampling farmland. When a mob walks over farmland, the game normally turns the farmland into dirt, destroying the crop. Seed Protect cancels that change, so the farmland stays as farmland and your crops remain safe.

### How It Works (for the user)
- Mobs cannot turn farmland into dirt when they walk over it.
- Players cannot accidentally trample newly planted crops.
- The change happens automatically; you don’t need to do anything extra.

### Features
- Works for all mobs and players.
- Compatible with vanilla farmland mechanics.

### Why Use This Mod?
- Keep your crops alive longer.
- Prevent accidental loss of planted seeds.
- Simple, no extra controls or items.
- Works on both single‑player and multiplayer worlds.

## What You'll Notice

- Mobs cannot trample farmland; farmland stays intact when mobs walk over it.

package com.seedprotect;

import net.minecraftforge.event.world.BlockEvent;
import net.minecraftforge.fml.common.Mod;
import net.minecraftforge.fml.common.eventhandler.SubscribeEvent;

@Mod(modid = SeedProtectMod.MOD_ID, name = SeedProtectMod.NAME, version = SeedProtectMod.VERSION)
@Mod.EventBusSubscriber(modid = SeedProtectMod.MOD_ID)
public final class SeedProtectMod {
    public static final String MOD_ID = "seedprotect";
    public static final String NAME = "Seed Protect";
    public static final String VERSION = "1.0.0";

    private SeedProtectMod() {
    }

    @SubscribeEvent
    public static void onFarmlandTrample(BlockEvent.FarmlandTrampleEvent event) {
        event.setCanceled(true);
    }
}
