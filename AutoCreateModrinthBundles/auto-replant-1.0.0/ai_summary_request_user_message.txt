# Auto Replant

This mod automatically replants certain crops and trees when you break them, using a seed or sapling from your inventory. No extra steps or items are required.

## How It Works (for the user)
- Breaking a fully grown wheat, carrot, potato, or beetroot block can cause the same crop to appear again after a short delay, using one seed from your inventory.
- Breaking a fully grown cocoa bean block (age 2 or more) can plant a new cocoa bean in the same spot.
- Breaking a log or log2 (tree trunk) can trigger a replant, though the exact behavior is limited.
- The replanting only happens on the server side and only if the block space is empty when the delay finishes.
- A tiny delay (about one tick, ~50 ms) is used before the new block is placed.

### Features
- Auto‑replants wheat, carrots, potatoes, beetroot, and cocoa beans.
- Handles cocoa beans and logs (trees) as well.
- Consumes the needed seed or sapling automatically; no manual planting required.
- Works on the server, so it affects all players on that server.
- No configuration or extra items needed.

### Why Use This Mod?
- Saves time by automatically replanting crops after you harvest them.
- Keeps your farm running without extra effort.
- Works with the same seeds you would normally use, so no extra items.
- Simple and straightforward—just break the block and let the mod do the rest.

    @SubscribeEvent

        World world = event.getWorld();
        BlockPos pos = event.getPos();

        EntityPlayer player = event.getHarvester();

        // Only run on server side
        if (world.isRemote || player == null) {
            return;

        if (block instanceof BlockCrops) {
            handleCropReplant(world, pos, state, block, player);
        }

        else if (block instanceof BlockCocoa) {
            handleCocoaReplant(world, pos, state, player);
        }
        // Handle saplings (trees)
        else if (block == Blocks.LOG || block == Blocks.LOG2) {
            handleTreeReplant(world, pos, player);
        }

package asd.itamio.autoreplant;

import net.minecraftforge.common.MinecraftForge;
