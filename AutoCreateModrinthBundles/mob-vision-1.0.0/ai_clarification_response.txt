GUARANTEED PLAYER-VISIBLE FACTS
- Hostile mobs within the configured range will chase and attack the nearest player.
- The range for each mob type can be set in the config file mobvision.txt.
- Setting a mob's range to 0 stops it from targeting players.
- The mod updates mob targeting once per second (every 20 game ticks).
- The config file is reloaded automatically when changed while the game runs.
- The mod works for all loaded hostile mobs such as zombies, skeletons, and spiders.
- No new items, blocks, or commands are added.

DO NOT CLAIM- Unclear: whether mobs can target players through solid blocks.
- Unclear: exact priority when multiple players are equally close.
- Unclear: how custom or mod‑added mobs are handled.
