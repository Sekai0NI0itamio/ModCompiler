# Sort Chest

## Description

**Sort Chest** is a lightweight client-side mod for Minecraft Forge 1.20.1 that adds a handy "Sort" button to every chest interface. With one click, it organizes your chest's contents by automatically combining partial stacks of the same item and grouping them together.

![Unsorted](https://cdn.modrinth.com/data/cached_images/ce26b2a2a2af0b8e296c78828a66ce2bd67116bc.png)
![When sorted](https://cdn.modrinth.com/data/cached_images/c07036c2389f54040ffc40aa6136f2ce7ceb1d26.png)

### How It Works (for the user)

- Open any chest (single, double, barrel, etc.).
- Look for the new **Sort** button in the chest GUI (usually near the top or bottom of the window).
- Click the button, and instantly:
  - All partial stacks of the same item are merged into full stacks (up to their maximum stack size, e.g., 64 for most items, 16 for snowballs/eggs, 1 for tools).
  - Items are moved so that identical items are grouped together in contiguous slots.
- Your chest becomes neatly organized without any manual dragging or rearranging.

### Features

- Works with all chest types: vanilla chests, trapped chests, barrels, and even modded chests that use the standard container GUI.
- Fully client-side: no server installation needed; works on any multiplayer server.
- One-click simplicity—no configuration required.
- Preserves stack size limits and respects item NBT data (so enchanted tools or named items won't merge incorrectly).
- Compatible with other inventory mods.
