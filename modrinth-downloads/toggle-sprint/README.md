# Modrinth Bundle: Toggle Sprint

Generated by the **Fetch Modrinth Project** GitHub Actions workflow.
This bundle contains all public metadata AND decompiled source for every
published version so an IDE agent can understand the mod offline.

## How to use this bundle

1. Read `summary.txt` for a full plain-text overview of the project.
2. Read `project.json` for the raw Modrinth API project object.
3. Read `index.json` for a machine-readable version list.
4. For each version, browse `versions/<id>/`:
   - `version.json`          full Modrinth version object
   - `changelog.txt`         changelog for this version
   - `files.json`            file list with download URLs and hashes
   - `<name>.jar`            the primary jar
   - `decompiled/mod_info.txt`        extracted mod metadata (key=value)
   - `decompiled/result.json`         full decompile result
   - `decompiled/src/main/java/...`   decompiled Java source tree
